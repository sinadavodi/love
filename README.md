# 💕 Romantic Surprise — Setareh Mishi

A cinematic romantic experience powered by sina davodi with this song  Behnam Bani's "Setareh Mishi".

## 🚀 How to use

### For the sender (you):
1. Open `index.html` (your GitHub Pages URL)
2. Type your girlfriend's name
3. Upload your photos together
4. Click **Generate Surprise Link**
5. Copy the link → send it via WhatsApp / Telegram / Instagram DM

### For the receiver (her):
1. She opens the link
2. Taps the screen
3. ❤️ Music plays, photos slideshow, lyrics sync — all automatically

## ✨ Features
- Cinematic Ken-Burns photo slideshow
- Auto colour theme per photo (10 emotional themes)
- Perfectly synced Persian lyrics (YouTube timestamps)
- Amiri calligraphy font
- Falling rose petals, bokeh, twinkling stars
- Click anywhere for heart burst
- Works on all devices — no app needed

## 📁 Files
- `index.html` — Creator page (sender uses this)
- `view.html`  — Romantic experience (girlfriend sees this)
- `.nojekyll`  — Required for GitHub Pages

## 🔧 GitHub Pages Setup
1. Push this folder to your GitHub repo
2. Go to repo **Settings → Pages → Source: Deploy from branch (main)**
3. Your creator URL: `https://YOUR-USERNAME.github.io/REPO-NAME/`
4. Links generated will point to: `https://YOUR-USERNAME.github.io/REPO-NAME/view.html#...`
